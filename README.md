# Secure Coding Practices Tutorial

Welcome to the Secure Coding Practices Tutorial repository! This project is designed to provide resources and materials related to secure coding practices, including presentations, notes, and threat modeling templates.

## Project Structure

The project is organized into the following directories:

- **docs/**: Contains instructional materials in various formats.
  - <a href="docs/AN4605.pdf" target="_blank">`AN4605.pdf`</a>: A PDF document related to secure coding practices.
  - <a href="docs/Day 2 - Overview of SDLC Lifecycle.pptx" target="_blank">`Day 2 - Overview of SDLC Lifecycle.pptx`</a>: A PowerPoint presentation covering the Software Development Life Cycle (SDLC).
  - <a href="docs/Information & Data Security Basics.pptx" target="_blank">`Information & Data Security Basics.pptx`</a>: A PowerPoint presentation on the basics of information and data security.
  - <a href="docs/IoT and IIoT Security.pptx" target="_blank">`IoT and IIoT Security.pptx`</a>: A PowerPoint presentation discussing security in IoT and IIoT contexts.
  - <a href="docs/Overview of Cyber Security attacks on source code.pptx" target="_blank">`Overview of Cyber Security attacks on source code.pptx`</a>: A PowerPoint presentation on cyber security attacks targeting source code.
  - <a href="docs/Secure Coding practices-Day 1.pptx" target="_blank">`Secure Coding practices-Day 1.pptx`</a>: A PowerPoint presentation from the first day of the secure coding practices session.
  - <a href="docs/Secure Coding practices-Day 2.pptx" target="_blank">`Secure Coding practices-Day 2.pptx`</a>: A PowerPoint presentation from the second day of the secure coding practices session.
  - <a href="docs/Secure SDLC.pptx" target="_blank">`Secure SDLC.pptx`</a>: A PowerPoint presentation on secure software development life cycle practices.
  - <a href="docs/Using cppcheck and Flawfinder for C & C++ Static.pptx" target="_blank">`Using cppcheck and Flawfinder for C & C++ Static.pptx`</a>: A PowerPoint presentation on using static analysis tools for C and C++.

- **media/**: Contains media files related to the session.
  - <a href="media/LINDDUN-threat-types.jpg" target="_blank">`LINDDUN-threat-types.jpg`</a>: An image file illustrating LINDDUN threat types.
  - <a href="media/Zero trust arc.jpeg" target="_blank">`Zero trust arc.jpeg`</a>: An image file related to the zero trust security model.

- **notes/**: Contains notes from the session.
  - <a href="notes/Ravana.docx" target="_blank">`Ravana.docx`</a>: A document containing notes or information related to the session.
  - <a href="notes/Secure Coding Session.txt" target="_blank">`Secure Coding Session.txt`</a>: A text file with notes from the secure coding session.

- **questions/**: Contains various question-related materials.
  - <a href="questions/Aristocract Questions.docx" target="_blank">`Aristocract Questions.docx`</a>: A document with questions related to the session.
  - <a href="questions/calculator.cpp" target="_blank">`calculator.cpp`</a>: A C++ code file that may serve as a template or example.
  - <a href="questions/format_string_arbitrary_write.cpp" target="_blank">`format_string_arbitrary_write.cpp`</a>: A C++ code file demonstrating a specific coding vulnerability.
  - <a href="questions/Questions.docx" target="_blank">`Questions.docx`</a>: A document containing additional questions.
  - <a href="questions/telnet-cooked (1).pcap" target="_blank">`telnet-cooked (1).pcap`</a>: A packet capture file related to telnet.
  - <a href="questions/BabyElephantWalk60 (1).wav" target="_blank">`BabyElephantWalk60 (1).wav`</a>: An audio file, possibly used as a sound effect or example.
  - <a href="questions/CoverSteg.png" target="_blank">`CoverSteg.png`</a>: An image file related to steganography.
  - <a href="questions/Olympus.jpg" target="_blank">`Olympus.jpg`</a>: An image file.

- **threat-modeling-templates/**: Contains templates for threat modeling.
  - <a href="threat-modeling-templates/az-security-threat-model.tm7" target="_blank">`az-security-threat-model.tm7`</a>: A threat modeling template for Azure security.
  - <a href="threat-modeling-templates/Azure Cloud Services.tb7" target="_blank">`Azure Cloud Services.tb7`</a>: A threat modeling template for Azure cloud services.
  - <a href="threat-modeling-templates/AzureTemplate.v3.tb7" target="_blank">`AzureTemplate.v3.tb7`</a>: Another version of an Azure threat modeling template.
  - <a href="threat-modeling-templates/Demo Threat Model.json" target="_blank">`Demo Threat Model.json`</a>: A JSON file containing a demo threat model.
  - <a href="threat-modeling-templates/MedicalDeviceTemplate.tb7" target="_blank">`MedicalDeviceTemplate.tb7`</a>: A threat modeling template for medical devices.
  - <a href="threat-modeling-templates/New Threat Model 1.tm7" target="_blank">`New Threat Model 1.tm7`</a>: A new threat model template.
  - <a href="threat-modeling-templates/New Threat Model.tm7" target="_blank">`New Threat Model.tm7`</a>: Another new threat model template.
  - <a href="threat-modeling-templates/new-model 1.json" target="_blank">`new-model 1.json`</a>: A JSON file for a new model.
  - <a href="threat-modeling-templates/new-model.json" target="_blank">`new-model.json`</a>: Another JSON file for a new model.

## Getting Started

To get started with the materials in this repository, simply navigate to the relevant directory and open the files of interest. The presentations and documents provide valuable insights into secure coding practices and related topics.
